# Focker

**Focker** — настольное приложение на **Python + PyQt6** для визуальной организации файлов, заметок и идей
в виде бесконечного холста.

Это не файловый менеджер и не заметочник в классическом смысле.
Это рабочее пространство, где ты **раскладываешь всё вручную**, как на реальном столе.

---

## Кратко о программе
-----------------------------------

**Focker** — это визуальная доска:

* перетаскивай файлы,
* создавай заметки,
* группируй объекты в области,
* работай с большим количеством данных без иерархий и папок.

Программа всегда под рукой:
* сворачивается в трей,
* открывается по глобальному хоткею,
* имеет мини-режим поверх всех окон.

---

## Основные возможности
-----------------------------------

* **Бесконечный холст**
  * Нет границ рабочей области
  * Масштабирование колесом мыши
  * Перемещение сцены мышью

* **Файлы**
  * Drag & Drop файлов прямо в окно
  * Предпросмотр изображений
  * Переименование без изменения оригинала
  * Открытие файла или папки из контекстного меню

* **Заметки**
  * Текстовые блоки прямо на холсте
  * Свободное изменение размера
  * Поддержка длинного текста

* **Групповые области**
  * Визуальные зоны для логического разделения
  * Можно использовать как фоны или категории

* **Контекстные меню**
  * Индивидуальные для объектов
  * Общее меню по пустому месту
  * Быстрое создание элементов

---

## Мини-режим
-----------------------------------

Отдельное компактное окно:

* список всех файлов с холста
* всегда поверх окон
* быстрый Drag & Drop файлов в другие программы

Удобно при работе в паре с редакторами, IDE или браузером.

---

## Управление
-----------------------------------

* ЛКМ — перемещение объектов  
* Колесо мыши — масштаб  
* ПКМ — контекстное меню  
* Ctrl+V — вставка из буфера  
  * файлы  
  * текст  
  * изображения  
* Drag & Drop — добавление файлов  

---

## Поведение окна
-----------------------------------

Важно:

* При нажатии на крестик **программа не закрывается**
* Она сворачивается в трей
* Полное завершение — только через иконку в трее

Это сделано намеренно.

## Настройки
-----------------------------------

* Цвет акцента
* Глобальный хоткей
* Папка хранения данных
* Автозапуск с Windows
* Повторный запуск гайда

Все настройки сохраняются автоматически.

---

## Хранение данных
-----------------------------------

* Состояние сцены сохраняется
* Используется простое шифрование
* Файлы не копируются, а привязываются по пути
* Вставленные изображения из буфера сохраняются локально
